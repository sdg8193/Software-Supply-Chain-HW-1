# Security Policy

## Supported Versions

Only the most recent version of this project, available on the `main` branch, is actively supported. Security patches will be applied to this version.

| Version | Supported          |
| ------- | ------------------ |
| Latest (Release 5.0) | :white_check_mark: |

## Reporting a Vulnerability

We take all security vulnerabilities seriously. If you discover a security issue, please report it by creating a GitHub issue in this repository or emailing sdg8193@nyu.edu for confidential issues.

Please include a detailed description of the vulnerability, steps to reproduce it, and any other relevant information that can help us understand and resolve the issue quickly. 

We appreciate your efforts to responsibly disclose your findings and will make every effort to acknowledge your contributions.
